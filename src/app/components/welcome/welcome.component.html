CRUD Operations On Form

  • In this project,We are performing CRUD operations on the details which we are getting from form
  • I have created the register form By using NGModel and performing two way binding
  • By filling all the details in the form ,there will be Save button by clicking on it,details get added.
  • The details which added in the form will get store in the side table.
  • My project is able to delete the details entered by the user ,if we user clicks on the delete button.
  • We can even edit the details of the user by clicking the edit button
